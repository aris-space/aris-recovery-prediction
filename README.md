# aris-recovery-prediction
Maturaarbeit Alexander Doongaji EMS Schiers 2020

Der Code wurde mit Hilfe der Spyder IDE geschrieben. Ein Python interpreter wird benötigt um den Code laufen zu lassen. 

Der Code kann ohne weitere Angaben ausgeführt werden. Alle Masseinheiten entsprechen den SI-Einheiten. Die Variabeln können direkt im Code verändert werden. 

Die auskommentierten Linien wurden zum Testen des Codes benötigt. 

Wenn man den Code ausführt, werden die bewegung in x und y Richtung aufgezeigt. Andere möglichkeiten sind auskommentiert daneben auskommenteirt aufgezeigt.
